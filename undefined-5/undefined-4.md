# 패스워드 분실

(1) 로그인 화면에서 패스워드 분실 버튼을 클릭합니다.

<figure><img src="../.gitbook/assets/2022-09-18 17_11_23-Book Management 외 페이지 69개 - 개인 - Microsoft​ Edge (1).png" alt=""><figcaption></figcaption></figure>

(2) 가입시 사용한 이메일을 입력하고 이메일 전송 버튼을 클릭합니다.

<figure><img src="../.gitbook/assets/2022-09-18 17_15_40-Book Management 외 페이지 70개 - 개인 - Microsoft​ Edge (1).png" alt=""><figcaption></figcaption></figure>

(3) 이메일로 받은 패스워드 변경용 링크를 클릭합니다.

<figure><img src="../.gitbook/assets/2022-09-18 17_13_28-패스워드 변경용 메일입니다 외 페이지 70개 - 개인 - Microsoft​ Edge (1).png" alt=""><figcaption></figcaption></figure>

(4) 링크를 클릭해서 들어온 화면에서 본인이 원하는 패스워드 값을 채우고 변경 버튼을 클릭합니다.

<figure><img src="../.gitbook/assets/2022-09-18 17_16_24-Book Management 외 페이지 71개 - 개인 - Microsoft​ Edge (2).png" alt=""><figcaption></figcaption></figure>

(5) 패스워드가 정상적으로 변경되면 변경이 되었다는 알림과 함께 로그인 화면으로 이동할 수 있는 버튼이 생깁니다. (실패 시, 붉은 화면과 함께 다시 시도해 달라는 메시지가 뜨게 됩니다.)

<figure><img src="../.gitbook/assets/2022-09-18 17_51_25-Book Management 외 페이지 72개 - 개인 - Microsoft​ Edge.png" alt=""><figcaption></figcaption></figure>
